# Курсовий проект: Автозаправна станція

Цей проект є курсовою роботою з розробки інформаційної системи для управління автозаправною станцією. Система забезпечує функціональність для управління типами палива, обліком користувачів та логуванням операцій.

## Опис

Система призначена для автоматизації процесів на автозаправній станції. Вона включає функції управління типами палива, реєстрації та автентифікації користувачів, а також ведення журналу операцій. Це забезпечує зручне керування та контроль діяльності на автозаправній станції.

## Функціональні можливості

1. **Логування користувачів**:
    - Форма для входу користувачів
    - Перевірка облікових даних користувача

2. **Управління типами палива**:
    - Перегляд всіх доступних типів палива
    - Додавання нових типів палива
    - Редагування існуючих типів палива
    - Видалення типів палива

3. **Логування операцій**:
    - Запис у журнал всіх дій з типами палива (вставка, оновлення, видалення)
    - Відображення історії операцій
